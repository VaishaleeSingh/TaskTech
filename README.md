# 🚀 TaskTech - Premium SaaS Customer Dashboard

[![Live Demo](https://img.shields.io/badge/demo-live-brightgreen.svg?style=for-the-badge)](https://techtask-1.netlify.app/)
[![Next.js](https://img.shields.io/badge/Next.js-14-black?style=for-the-badge&logo=next.js)](https://nextjs.org/)
[![Tailwind CSS](https://img.shields.io/badge/Tailwind_CSS-v4-38B2AC?style=for-the-badge&logo=tailwind-css)](https://tailwindcss.com/)

A **pixel-perfect, production-ready** replica of a high-end SaaS customer dashboard. This project demonstrates advanced frontend engineering techniques using **Next.js 14**, **TypeScript**, and **Tailwind CSS**.

### 🔗 [View Live Deployment](https://techtask-1.netlify.app/)

---

## ✨ Key Features

- **🎯 Pixel-Perfect Design**: Meticulously crafted to match professional SaaS UI/UX specifications.
- **📱 Fully Responsive**: Seamless experience across mobile, tablet, and ultra-widescreen monitors.
- **🎭 Advanced Animations**: 
  - Staggered fade-and-slide entry for all dashboard sections.
  - Smooth progress bar transitions using native CSS.
  - Self-drawing SVG handwritten annotations and pulsating badges.
  - Dynamic 3D overlapping card layout in the header.
- **🧩 Modular Architecture**: Clean, reusable React components built for scalability.
- **⚡ Performance Optimized**: Minimal dependencies, leveraging Next.js 14 App Router and server-side optimizations.

## 🛠️ Tech Stack

- **Framework**: [Next.js 14 (App Router)](https://nextjs.org/)
- **Language**: [TypeScript (Strict)](https://www.typescriptlang.org/)
- **Styling**: [Tailwind CSS v4](https://tailwindcss.com/)
- **State Management**: Native React Hooks
- **Icons & Assets**: Scalable SVG Integration

## 📸 Dashboard Sections

### 1. Unified Top Section
A sophisticated overlapping header featuring:
- A full-width blue primary background.
- Floating Intercom customer profile card.
- Tilted Company Activation card with real-time animating progress trackers.

### 2. Celebrate Milestones
A vibrant purple panel displaying recent customer achievements, integrating real-world assets from **Slack** and **Attio**.

### 3. Automatic Enrichment
AI-powered data profile section featuring a clean, column-based table with "colliding" branding icons.

### 4. CRM Integration
Real-time CRM sync table demonstrating data connectivity with **HubSpot**, **Intercom**, and **Attio** brand assets.

### 5. Smart Annotations
Handwritten-style "Also done for you" overlays that simplify complex technical status into user-friendly insights.

## 🚀 Getting Started

### Prerequisites
- Node.js 18.x or later
- npm or yarn

### Installation
1. **Clone the repository:**
   ```bash
   git clone https://github.com/VaishaleeSingh/TaskTech.git
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Run the development server:**
   ```bash
   npm run dev
   ```

4. **Open your browser:**
   Navigate to [http://localhost:3000](http://localhost:3000)
